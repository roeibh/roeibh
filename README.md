### Hi, I'm Roei 👋

I work on **AppSec × AI** — agentic systems that reason about vulnerabilities instead of
pattern-matching them. Mostly TypeScript, Node, and AWS, with a past life in security research
and automation.

Most of my day job lives in private repos, so what's here is the other half: the tooling I build
to make working with coding agents less annoying.

---

### 🤖 Claude Code skills & plugins

| | |
| --- | --- |
| **[human-comments](https://github.com/roeibh/human-comments)** | Stops agents writing essay comments. Almost none survive, and the ones that do are fragments a senior engineer would actually leave. Ships as a Claude skill + Cursor rule. |
| **[interactive-spec](https://github.com/roeibh/interactive-spec)** | Turns a written spec into a self-contained interactive HTML mockup you can click through — then sends your inline comments back to Claude. A closed review loop, not a one-shot mockup. |
| **[marp-deck-architect](https://github.com/roeibh/marp-deck-architect)** | Builds Marp decks. Four archetypes (Builder, Operator, Storyteller, Educator) that shape not just the theme, but how the deck gets written. |
| **[interviewer-kit](https://github.com/roeibh/interviewer-kit)** | Engineering interview prep — personalizes interviews per candidate and turns a job description into a reusable interview template. |
| **[morning-briefing](https://github.com/roeibh/morning-briefing-claude-plugin)** | Triages your inbox, calendar, and industry noise into one briefing with draft responses and ranked priorities. |
